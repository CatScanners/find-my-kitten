## Minimum personal computer requirements
- docker (installation instructions can be found at: https://docs.docker.com/engine/install/ubuntu/)
- Linux or WSL

### Simulator requirements
- Ubuntu 24, aka Ubuntu Jazzy (VMs wont be enough, you need a full dual boot)
- 150 GB of storage (excl. the OS)
- Nvidia GPU (required for simulator tests)
- python3 version 3.12.4 or newer


## drone requirements
- Ndivia Jetson -board
- If you are already using a Nvidia Jetson -computer, please refer to the [Nvidia Jetson initialization](#jetson-initialization).
